What is the difference between map, filter, forEach and find ?

  • ForEach: It is used when we need to execute a function for individual element of an array. It does not return a new array based on the given array.
  • Map: It is used to transform every element of an array by calling a function and return a new array with the elements in the same order as the original array.
  • Filter: It is used to return a new array with only the elements that meet the condition specified in a function.
  • Find:It is used to return the first element in the array that satisfies the specified condition. It returns null if no such element is found.

    What is event loop and how does it work?

    The event loop is a mechanism in JavaScript for handling asynchronous operations. Although JavaScript executes all operations on a single thread, it gives us the feel of multi-threading.

    The call stack keeps track of all the operations in the line to be executed. When a function is done executing, it is popped from the stack.

    The event queue sends new functions to the stack for processing, It is like the queue data structure to maintain the sequential execution of the operations.

    Event loop allows JavaScript to handle multiple concurrent events efficiently, making it suitable for building responsive and interactive web applications.
